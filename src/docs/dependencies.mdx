<section>
    <header>
    ## Styling dependencies
    These libraries are essential and used in almost every component here.
    </header>

    - [**Tailwind CSS**](https://tailwindcss.com/) - A utility-first CSS framework for styling components with ease.
    - [**tailwind-merge**](https://github.com/dcastil/tailwind-merge) - A utility to intelligently merge Tailwind class names, preventing conflicts.
    - [**clsx**](https://github.com/lukeed/clsx) - A small utility to conditionally join class names dynamically.
    - [**Class Variance Authority**](https://cva.style/) - A utility to conditionally generate and manage Tailwind class names with variants support in a clean, scalable way.
    
</section>

<section>
    <header>
        ## Animation dependencies
        Some components require animation libraries to achieve complex interactions and motion effects.
    </header>

    - [**Motion**](https://motion.dev/) - A powerful animation library for React.
    - [**React Three Fiber**](https://r3f.docs.pmnd.rs/) - React renderer for Three.js, enabling 3D scenes with JSX and seamless React integration for interactive experiences.
</section>

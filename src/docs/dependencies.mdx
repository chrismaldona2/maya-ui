<section>
    <header>
    ## Styling dependencies
    These libraries are essential and used in almost every component here.
    </header>

    - [**Tailwind CSS**](https://tailwindcss.com/) - Utility-first CSS framework with atomic class patterns for rapid styling.
    - [**tailwind-merge**](https://github.com/dcastil/tailwind-merge) - A tool for merging Tailwind class names efficiently, preventing style conflicts.
    - [**clsx**](https://github.com/lukeed/clsx) - A compact library for conditionally combining class names.
    - [**Class Variance Authority**](https://cva.style/) - Variant-based class management with type-safe configurations.
    
</section>

<section>
    <header>
        ## Animation dependencies
        Some components require animation libraries to achieve complex interactions and motion effects.
    </header>

    - [**Motion**](https://motion.dev/) - Production-ready animations with spring physics and gesture support.
    - [**React Three Fiber**](https://r3f.docs.pmnd.rs/) - React-powered Three.js wrapper for building interactive 3D content.

</section>
